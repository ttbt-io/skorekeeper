# Game Scenario 2: Rockets vs Aviators (Mercy Rule & ITB)

**Teams:**
*   **Away:** Rockets (12 Players)
*   **Home:** Aviators (12 Players)

**Rules:**
*   **Mercy Rule:** Maximum 3 runs per half-inning. If a play results in more than 3 runs total for the inning, the scorekeeper must manually adjust the inning score to 3.
*   **Tie Breaker (ITB):** Starting in the 6th inning, the last batter from the previous inning starts on 2nd base.
*   **Roster:** Continuous batting order (all 12 players bat).
